## 🎯 Why this repo / 為什麼有這個 repo

- 把每一關當作一篇學習日誌，逼自己**真的搞懂**，而不是查到答案就過。
- Practice writing things down in public. If I can't explain a level, I haven't learned it.
- 給後來的人一個「不直接洩題、但能看到思考路徑」的學習參考。
- 密碼預設用 `<details>` 摺疊，**請先自己試**再展開驗證。
